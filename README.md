# 🛡️ CyberSentinel AI Dashboard

CyberSentinel is an AI-powered cybersecurity simulation that detects system issues and recommends automated fixes in real time.

---

## 🚀 What it does

* Monitors system health (CPU, memory, errors)
* Detects critical issues
* Suggests intelligent recovery actions
* Displays everything in a live dashboard

---

## 🧠 AI Logic

| Problem         | Action            |
| --------------- | ----------------- |
| High CPU        | `scale_up`        |
| Memory leak     | `restart_service` |
| Too many errors | `block_ip`        |

---

## 🎯 Features

* Real-time dashboard (React + Tailwind)
* FastAPI backend with live API
* OpenEnv-compatible environment
* Rule-based AI decision engine
* Dynamic alerts + AI actions

---

## ⚙️ Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** FastAPI
* **AI Logic:** Python (rule-based inference)

---

## ▶️ Run Project

### Backend

```bash
python main.py
```

### Frontend

```bash
npm run dev
```

## 🔥 Why it’s powerful
